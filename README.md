MERN Book App

Bu proje, MERN stack (MongoDB, Express, React, Node.js) kullanılarak oluşturulmuş basit bir kitap yönetim uygulamasıdır. Amaç, full-stack geliştirme sürecini test etmek ve uçtan uca çalışan bir CRUD yapısı kurmaktır.

Proje deneme ve öğrenme amaçlıdır.

Amaç

Backend ve frontend arasındaki veri akışını test etmek

REST API yapısını gerçek bir uygulama üzerinde denemek

React tarafında API kullanımı ve sayfa akışını görmek

Full-stack bir projenin temel yapı taşlarını bir araya getirmek

Kullanılan Teknolojiler Backend

Node.js

Express.js

MongoDB

Mongoose

Frontend

React

React Router

Axios

Tailwind CSS

Özellikler

Kitap ekleme

Kitap listeleme

Kitap detaylarını görüntüleme

Kitap güncelleme

Kitap silme

Proje Durumu

Uygulamanın temel yapısı kurulmuştur. Şu anki haliyle bir test ve öğrenme projesidir.

İlerleyen süreçte hem backend hem frontend tarafında geliştirmeler yapılması planlanmaktadır.

Not

Bu proje, üretim ortamı için değil; öğrenme, deneme ve pratik yapma amacıyla oluşturulmuştur.
